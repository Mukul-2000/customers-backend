# Customers Backend

REST API built with Node.js, Express, TypeScript and MongoDB

## Features
- Create, Read, Update, Delete customers
- MongoDB integration
- TypeScript support

## Tech Stack
- Node.js
- Express.js
- MongoDB
- TypeScript

## Setup
npm install
npm run dev

## API Endpoints
GET    /customers
POST   /customers
PUT    /customers/:id
DELETE /customers/:id
